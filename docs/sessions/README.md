# Session Logs

This directory contains summaries of AI agent work sessions on the Moddy project.

## Purpose

Each session log provides a detailed record of what was done, enabling continuity across conversations and helping track the evolution of the project.

## File Naming Convention

```
YYYY-MM-DD_short-description.md
```

Examples:
- `2026-03-21_docs-reorganization.md`
- `2026-03-22_starboard-bug-fix.md`

## Template

```markdown
# Session: [Short Description]

**Date:** YYYY-MM-DD
**Agent:** [Claude Code / other]

## Summary

Brief overview of what was accomplished.

## Changes Made

- `path/to/file.py` — Description of change
- `path/to/other.py` — Description of change

## Decisions & Rationale

- Decision 1: Why it was made
- Decision 2: Why it was made

## Known Issues / Follow-ups

- [ ] Any remaining work
- [ ] Things to watch out for
```
